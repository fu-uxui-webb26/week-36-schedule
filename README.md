## Introduktion

Hur skapar man digitala upplevelser som faktiskt löser riktiga problem? Den här veckan introducerar vi UX, UI och Design Thinking – en användarcentrerad process för att förstå människor och deras behov innan vi börjar designa lösningar.

Vi fokuserar på de två första stegen i Design Thinking: Empathize och Define. Genom intervjuer, research och analys lär vi oss att identifiera användarnas utmaningar och formulera tydliga problem som är värda att lösa.

Under veckan startar även kursens grupprojekt, där ni kommer att följa hela Design Thinking-processen från första användarintervju till färdig prototyp.
